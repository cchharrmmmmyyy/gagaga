# Pixel Pong Adventure

Pixel Pong Adventure turns classic Pong into an original 8/16-bit platform arena game. It keeps the paddle rally core, then adds quest worlds, local versus play, power-ups, player skills, arena hazards, multi-ball chaos, and a boss round.

## Features

- Quest mode with 5 themed worlds: grass, cave, water, sky, and castle.
- Local versus mode for two players on one keyboard.
- Power-ups: wide paddle, fast ball, slow ball, fire ball, shield, ghost ball, multi-ball, and coin score.
- Skill charge system: rallies build charge, then players can fire a power shot and shield burst.
- Arena hazards: breakable blocks, bumpers, water gates, moving cloud platforms, flames, and a boss trick orb.
- Fully canvas-drawn pixel backgrounds, UI, paddles, balls, obstacles, and effects.

## Controls

- Enter: start, serve, or continue.
- Quest mode: W/S or arrow keys move P1, Space or F uses skill.
- Versus mode: P1 uses W/S + F, P2 uses arrow keys + /.
- Mouse or touch: drag P1 paddle, click or tap to start/serve.
- P: pause or resume.
