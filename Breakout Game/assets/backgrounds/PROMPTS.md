# Background Asset Prompts

These PNG files are lightweight local placeholders so the game runs immediately.
They can be replaced with higher-detail generated art later.

Use this direction for replacement assets:

- Original bright platform-adventure game background, no Nintendo characters, no logos, no trademarked elements, no text.
- 800x600 landscape game backdrop, clear playable foreground, soft parallax-friendly depth.
- Cartoon illustration style, polished casual arcade game, rounded shapes, readable color contrast.

Themes:

- `grassland.png`: sunny meadow, rolling green hills, blue sky, soft clouds.
- `cavern.png`: friendly underground cave, warm rock platforms, glowing crystals, safe arcade mood.
- `lagoon.png`: tropical shell lagoon, turquoise water, beach colors, gentle waves.
- `skyway.png`: cloud walkway high in the sky, bright blue atmosphere, floating soft cloud forms.
- `castle.png`: playful fantasy castle approach, warm stone, dramatic but family-friendly.
